1) what is method overloading? <br>
   a)  Methods within a class can have the same name if they have different parameter lists then those methods are called overloaded methods. <br>
   b) The compiler does not consider return type when differentiating methods, so you cannot declare two methods with the same signature even if they have a different return type. <br>

2) what is variable arguments? <br>
   a) To use varargs, we follow the type of the last parameter by an ellipsis (three dots, ...), then a space, and the parameter name. <br>
   eg:  public Polygon polygonFrom(Point... corners)  <br>
