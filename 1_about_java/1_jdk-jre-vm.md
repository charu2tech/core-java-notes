2) can i execute java program in one step? <br>
   In JDK 11, Java introduced the ability to launch a single-file source-code program with the java launcher, now multi-file as of JDK 22, without first needing to explicitly compile the source code. This works by the java launcher automatically invoking the compiler and storing the compiled code in-memory. <br>
